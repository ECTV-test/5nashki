Flip Tiles — Scenes (v3)

РЕДАГУЙТЕ ТІЛЬКИ config.js.

РЕКОМЕНДОВАНІ РОЗМІРИ ЗОБРАЖЕНЬ
- Фон (bg): 1920×1080 (найкраще) / 2560×1440 (дуже чітко) / 3840×2160 (4K, важче для завантаження)
- Картинки для квадратів (front): 800×800 або 1024×1024 (найкраще). Мінімум 600×600

ОРГАНІЗАЦІЯ ФАЙЛІВ (щоб не плутатися)
- Кожну сцену зберігайте в окремій папці:
  assets/scene01/bg.jpg + tile1..tile5.jpg
  assets/scene02/bg.jpg + tile1..tile5.jpg
- Те, що повторюється між сценами, кладіть у:
  assets/shared/
  і підставляйте шлях у сцені.

ТАЙМІНГИ
- timing.holdTileMs: скільки тримаємо EN і скільки тримаємо UA для КОЖНОГО квадрата (послідовно).
- timing.holdFinalMs: скільки тримаємо ВСІ квадрати на EN, потім ВСІ на UA.
- timing.holdFrontBetweenScenesMs: після фінального UA повертаємо квадрати на картинки (front) і чекаємо перед наступною сценою.

ШВИДКОСТІ (для кожного квадрата окремо)
- flipToBackMs: швидкість першого перевороту (КАРТИНКА/front → EN/back).
- flipTextMs: швидкість перевороту тексту на звороті (EN ↔ UA) БЕЗ повернення до картинки.

КЕРУВАННЯ / HUD
- Автозапуск при відкритті сторінки
- Тап/клік по екрану перезапускає анімацію (коли HUD прихований)
- HUD прихований за замовчуванням:
  - подвійний тап/подвійний клік у ЛІВОМУ ВЕРХНЬОМУ куті показує/ховає HUD
